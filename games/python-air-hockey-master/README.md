A fun air hockey game built in python using [pyxel](https://github.com/kitao/pyxel)!

### Controls

    self.p1KeyUp = pyxel.KEY_W
    self.p1KeyDown = pyxel.KEY_S

    self.p2KeyUp = pyxel.KEY_UP
    self.p2KeyDown = pyxel.KEY_DOWN
